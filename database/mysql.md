##### 存储引擎对比 

> MyISAM 可被压缩，存储空间小，在筛选大量数据时非常快 
>
> InnoDB 需要更多存储空间，会在内存中建立专用的缓冲池用于高速缓冲数据跟索引 

|              | MyISAM | InnoDB |
| ------------ | ------ | ------ |
| 事务         | 不支持 | 支持   |
| 外键         | 不支持 | 支持   |
| 行级锁       | 不支持 | 支持   |
| 自动奔溃恢复 | 不支持 | 支持   |
|              |        |        |
