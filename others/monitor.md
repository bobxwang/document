##### 四个黄金指标

- 错误
  - 基础 
    - 宕机，端口跟进程存活
  - 业务 
    - 核心功能错误 
    - 基础数据丢失
    - 可用实例占比
    - 上游依赖错误 
    - 日志错误
- 延迟
  - 基础 
    - IO Wait
    - 网络延迟 
  - 业务 
    - 核心功能延迟
    - 集群数据同步延迟
- 流量
  - 基础 
    - 磁盘/网卡流量
  - 业务 
    - 核心功能请求速率
- 饱和度
  - 基础
    - 基础资源使用率
    - 打开文件的句柄数
    - TCP 连接数
  - 业务
    - 核心功能使用率
    - 集群数据均衡度
    - 队列长度 

##### 监控

- 墨盒监控
  - 外部端口探活
  - HTTP 探测
  - 端到端功能监控
- 白盒监控
  - 日志 
  - 命令行
  - 主动上报
  - API 
    - REST
    - JMX
    - OpenMetrics 

##### 监控分类

- 端用户体验监控
  - 性能
  - 地区/城市 
  - 运营商
  - 版本
- 业务监控
  - 核心指标
  - 登录注册
  - 下单支付
- 应用层监控
  - URL
  - Service
  - Cache
  - Sql
  - QPS
- 系统层监控
  - 物理机/虚拟机/OS
  - CPU/Memory/NetWork
- 基础设施层监控
  - 网络/交换机
  - 网络流量/丢包/错包
  - 连接数
